# Quản lý bản tin truyền hình

## Các lớp

### interface New

- BanTin
- QuangCao
- LiveStream

### interface PeoPle

- Anchor
- Author
- Reporter

### Quản lý thời gian

- Calendar
- CalendarDay
- SetTime

### Quản lý kênh / thể loại

- Category
- Channel

## Chức năng

**1. Quản lý bản tin**
   - Thêm bản tin
   - Đặt thể loại / tác giả

**2. Quản lý kênh**
   - Thêm kênh
   - Danh sách kênh

**3. Quản lý thời gian biểu**
   - Đặt thời gian

**4. Quản lý tác giả**
   - Tạo mới
   - Quản lý bản tin của tác giả

**5. Thể loại**
   - Tạo mới
   - Quản lý bản tin của thể loại đó

**6. Xuất kết quả theo thời gian**
   - Chọn ngày
   - Chọn khung thời gian
